Documentação das Funções PlaceIn

Variáveis Globais

$MT_raiz

Contém o domínio do cliente. Ideal para uso quando é preciso informar o endereço completo em um script.

Funções

NTdifdata($inicio, $fim)

Descrição: Retorna a diferença de dias entre duas datas. Pode aceitar a data em vários formatos (unix ou placein), ex: 2015-08-01, 01/08/2015, 01-08-2015.

Parâmetros:

  • $inicio: Data inicial.
  • $fim: Data final.

kripto_in($texto, $chave)

Descrição: Criptografa uma informação que poderá ser descriptografada com kripto_out($textocriptografado, $chave). A $chave deve ser um valor numérico e deve ser informada igual no in e no out.

Parâmetros:

  • $texto: Texto a ser criptografado.
  • $chave: Chave de criptografia.

proximo_dia_util($data, $ordem)

Descrição: Envia a data do próximo dia útil após uma data informada. A data pode ser inserida no padrão yyyy-mm-dd ou dd-mm-yyyy e será retornada no mesmo formato. A $ordem pode ser um número indicando a quantidade de dias úteis para frente ou para trás (negativo), ou "ontem" para o dia útil anterior.

Parâmetros:

  • $data: Data de referência.
  • $ordem: Quantidade de dias úteis ou "ontem".

filemtime_remote($url)

Descrição: Checa a data de modificação de um arquivo, mesmo que esteja em um servidor externo.

Parâmetros:

  • $url: URL do arquivo.

protegeNT($url)

Descrição: Interrompe a execução do script e direciona para a URL indicada se o usuário não estiver logado no PlaceIn.

Parâmetros:

  • $url: URL de redirecionamento.

NTacentos($texto)

Descrição: Ajusta acentos de outras codificações para o padrão PlaceIn.

Parâmetros:

  • $texto: Texto a ser ajustado.

baseHeader()

Descrição: Exibe os headers básicos - utf8, bibliotecas js, etc.

NTendereco($retorno)

Descrição: Captura dados de endereço da planilha Parametrização. O retorno padrão é um echo em UTF8, mas pode ser alterado para ISO ou return.

Parâmetros:

  • $retorno: Formato de retorno (UTF8, ISO, return).

NTflush()

Descrição: Força o envio de um echo do PHP. Colocar logo após o echo().

urlraiz()

Descrição: Retorna a raiz da URL presente no navegador. Ex: www.k2web.com.br.

consultaA($psql, $cache, $encode, $cryptografia, $executarURL)

Descrição: Executa o NTSELECT informado em $psql, retornando um array em UTF8. Pode utilizar cache para agilizar a consulta.

Parâmetros:

  • $psql: Comando NTSELECT.
  • $cache: "sim" ou "não" para usar cache.
  • $encode: Formato de codificação (UTF8, ISO).
  • $cryptografia: "sim" ou "não" para usar criptografia.
  • $executarURL: "sim" ou "não" para executar URLs dentro do conteúdo.

MT_relatorio($arte)

Descrição: Busca dados de abertura da arte MailTo especificada e retorna um array no formato [email[ip][hora].

Parâmetros:

  • $arte: Identificador da arte MailTo.

QRcode($link, $echo)

Descrição: Gera um arquivo QRcode com nome qrpng.png na raiz. Se $echo for "sim", exibe o arquivo em PNG no navegador.

Parâmetros:

  • $link: Link a ser codificado.
  • $echo: "sim" para exibir o QRcode.

NTecampoS($string, $start, $end)

Descrição: Extrai dados de uma string, entre os caracteres $start e $end.

Parâmetros:

  • $string: Texto a ser processado.
  • $start: Caracter inicial.
  • $end: Caracter final.

mergelistas($lista1, $lista2, $separador, $diferemaiuscula)

Descrição: Compara duas listas, incluindo na $lista1 os itens que estão na $lista2 e não estão na $lista1. Indique o separador da lista e se diferencia maiúsculas de minúsculas.

Parâmetros:

  • $lista1: Primeira lista.
  • $lista2: Segunda lista.
  • $separador: Separador de itens.
  • $diferemaiuscula: "s" ou "n" para diferenciar maiúsculas de minúsculas.

login($link, $linkerro, $planilha, $labelenviar, $divisor)

Descrição: Gera um formulário de login.

Parâmetros:

  • $link: Chave do apoiophp de destino após o login.
  • $linkerro: URL ou chave do apoiophp em caso de erro.
  • $planilha: Planilha alternativa de login.
  • $labelenviar: Texto do botão enviar.
  • $divisor: Tag HTML para separação (default é
    ).

logout($variaveis)

Descrição: Apaga os cookies e faz logout. $variaveis deve ser a página de destino após o logout.

Parâmetros:

  • $variaveis: Página de destino.

protege($variaveis)

Descrição: Cancela a execução do script se o usuário não estiver logado e direciona para a página determinada em $variaveis.

Parâmetros:

  • $variaveis: Página de destino.

user($variaveis)

Descrição: Recupera o cookie de login usuNT. Se $variaveis for "NT", mostra o login via PlaceIn.

Parâmetros:

  • $variaveis: Modo de recuperação.

geraform2($destinatarios, $assunto, $camposform, $populate, $grupo, $redirect, $objeto, $labelenviar, $retorno, $msg_aguarde, $tipolabel, $css, $msgok, $politica, $utf, $modo, $step)

Descrição: Gera um formulário padrão Geraform. Deve incluir a função baseHeader() antes da linha onde está a função geraform2(). Permite customização de visual, validações de obrigatoriedade, e pode ser utilizado como formulário simples ou avançado.

Parâmetros:

  • $destinatarios: Destinatários do formulário.
  • $assunto: Assunto do formulário.
  • $camposform: Campos do formulário.
  • $populate: Dados a serem preenchidos no formulário.
  • $grupo: Grupo a ser gravado no MailTo.
  • $redirect: URL de redirecionamento em caso de sucesso.
  • $objeto: Nome da div para o formulário.
  • $labelenviar: Texto do botão enviar.
  • $retorno: "echo" ou "return".
  • $msg_aguarde: Mensagem de aguarde.
  • $tipolabel: Tipo de label.
  • $css: Caminho do CSS.
  • $msgok: Mensagem de sucesso.
  • $politica: Mostrar link para política de privacidade.
  • $utf: Codificação dos acentos.
  • $modo: Modo de encurtar o nome das variáveis.
  • $step: Habilita a quebra do formulário em páginas.
Ler Mais/Ler Menos

Utilização avançada: você pode interagir com os dados do formulário colocando na variável $populate os dados que deseja preencher no formulário (no padrão @/). Pode também direcionar o envio do formulário para uma $chave do apoiophp, colocando o nome da chave na variável $grupo. Dessa forma o Geraform enviará as seguintes variáveis para a chave do apoiophp: $destinatarios,$assunto,$camposform,$redirect e uma variável $dados com a massa de dados no padrão @/. Você deve retornar um echo para o formulário com uma string iniciada com OK| em caso de sucesso e erro| no caso de erro. Ex: "OK| seus dados foram enviados" ou "erro|O e-mail informado já existe". Você pode devolver comandos html ou scripts nesse retorno. $retorno: o padrão é "echo" e imprime o formulário na tela (não deixe essa variável em branco). Se for return devolve o formulário em string (ideal para manipular o código do formulário dentro do apoiophp) $msg_aguarde: coloque aqui a frase que quer que apareça quando o formulário estiver sendo enviado. $tipolabel: se for placeholder coloca o nome do campo dentro do próprio campo. $css: indique aqui o caminho completo do css. Ex: /htmlf/MailTo/geraform.css (note que nesse CSS existe tratamento do box de erro. Se comentar ou alterar esse CSS tenha certeza de manter os códigos necessários para que o box de texto do retorno de possível erro funcione). $msgok: altere aqui a mensagem de retorno - enviado com sucesso $politica: mostra o link para acesso a política de privacidade: /politicadeprivacidade_geral.html para desativar coloque "não" $utf: encode dos acentos. O default é utf-8, a outra opcão é "iso" $modo: modo de encurtar o nome das variáveis pode ser "" ou "n" (o primeiro substitui as vogais por vazio e o segundo por números 1,2,3,4,5 $step: habilita a quebra do formulário em páginas. Para isso mude para "on" e coloque o código |1,stepfim|2,stepinicio| onde deseja quebrar a pagina no geraform IMPORTANTE: o Graform tem um código no CSS padrão que mostra um box de texto com possíveis erros no preenchimento. Ao alterar o CSS tenha cuidado de manter esses códigos. 

$destinatarios="";
$assunto="";
$camposform="";
$populate="";
$grupo="";
$redirect="";
$objeto="";
$labelenviar="Enviar";
$retorno="echo";
$msg_aguarde="Por favor aguarde, enviando dados";
$tipolabel="";
$css="/htmlf/paginas/geraform.css";
$msgok="Mensagem enviada com sucesso.";
$politica="sim";
$utf="utf-8";
$modo="";
$step="off";

baseHeader();
geraform2($destinatarios,$assunto,$camposform,$populate,
            $grupo,$redirect,$objeto,$labelenviar,$retorno,
            $msg_aguarde,$tipolabel,$css,$msgok,$politica,
            $utf,$modo,$step);

NTcurl($url) ou NTcurlP($url)

Descrição: Acessa um endereço HTTP enviando e recebendo variáveis. A versão NTcurl envia como GET e a NTcurlP como POST.

Parâmetros:

  • $url: URL a ser acessada.

unzipfotos($arquivo, $ncontrole)

Descrição: Descompacta um arquivo zip e inclui em uma notícia do PlaceIn. Informe o caminho correto do arquivo zip e um $ncontrole válido.

Parâmetros:

  • $arquivo: Caminho do arquivo zip.
  • $ncontrole: Número de controle.

zipafotos()

Descrição: Faz download da pasta biblioteca/fotos em formato zip.

NTexcel($arquivo, $dados)

Descrição: Envia dados para download no formato Excel. Os dados devem ser separados no padrão }| - campos separados por } e linhas por |.

Parâmetros:

  • $arquivo: Nome do arquivo.
  • $dados: Dados a serem incluídos.

emostra($variaveis)

Descrição: Mostra dados em modo echo.

NT_relatorionews($campos, $filtro)

Descrição: Função interna que exibe a div com os valores da busca de qualquer campo do formato inserirNT('nomecoluna', valor);.

Parâmetros:

  • $campos: Campos a serem mostrados.
  • $filtro: Filtro de busca.

MTenvia($destinatarios, $assunto, $mensagem, $remetente, $grupo)

Descrição: Função para envio de e-mail. O grupo é o grupo a ser gravado no MailTo. Se o destinatário contiver e-mails do próprio domínio ou da K2web, será enviado em anexo a auditoria Feedback.

Parâmetros:

  • $destinatarios: E-mails dos destinatários.
  • $assunto: Assunto do e-mail.
  • $mensagem: Corpo do e-mail.
  • $remetente: E-mail do remetente.
  • $grupo: Grupo a ser gravado no MailTo.

MTenvia2($destinatarios, $assunto, $mensagem, $remetente, $grupo, $server)

Descrição: Função para envio de e-mail via MailTo utilizando o remetente indicado em $server. Especificar o servidor da seguinte forma: smtp|porta|remetente|senha. Exemplo: mail.k2web.com.br|587|teste@k2web.com.br|teste4321|true|ssl.

Parâmetros:

  • $destinatarios: E-mails dos destinatários.
  • $assunto: Assunto do e-mail.
  • $mensagem: Corpo do e-mail.
  • $remetente: E-mail do remetente.
  • $grupo: Grupo a ser gravado no MailTo.
  • $server: Configuração do servidor SMTP.

MTenviaA($destinatarios, $assunto, $mensagem, $remetente, $grupo, $empresa)

Descrição: Função que grava diretamente no sistema de mensagens do AndroidMail, no domínio do e-mail, em vez de enviar um e-mail.

Parâmetros:

  • $destinatarios: E-mails dos destinatários.
  • $assunto: Assunto da mensagem.
  • $mensagem: Corpo da mensagem.
  • $remetente: E-mail do remetente.
  • $grupo: Grupo a ser gravado no MailTo.
  • $empresa: Empresa relacionada.

gravaMSG($Empresa, $Ref, $Texto, $Destinatarios, $Remetente, $Assunto, $Mensagem, $planilha, $data_ativacao, $flag, $prazo)

Descrição: Função para gravar mensagem diretamente no AndroidMail.

Parâmetros:

  • $Empresa: Nome da empresa.
  • $Ref: Referência.
  • $Texto: Texto da mensagem.
  • $Destinatarios: E-mails dos destinatários.

  • $Assunto: Assunto da mensagem.
  • $Mensagem: Corpo da mensagem.
  • $planilha: Planilha associada.
  • $data_ativacao: Data de ativação.
  • $flag: Flag.
  • $prazo: Prazo.

NTnomearquivo($texto, $separador)

Descrição: Limpa string para ser utilizada como nome de arquivo, eliminando acentos, caracteres especiais, etc. O separador padrão é _.

Parâmetros:

  • $texto: Texto a ser limpo.
  • $separador: Separador para o nome do arquivo.

Descrição: Extrai o valor de um campo padrão @/ em uma massa de dados.

  • $massa: Massa de dados.

data2Unix($entdata)

Descrição: Converte data para o padrão Unix.

Parâmetros:

  • $entdata: Data a ser convertida.

texto2Unix($texto)

Descrição: Converte texto para o padrão Unix. Exemplo: & muda para @e@.

  • $texto: Texto a ser convertido.

Unix2data($entdata)

Descrição: Converte data no formato YYYY-mm-dd para dd-mm-YYYY.

Parâmetros:

  • $entdata: Data a ser convertida.

Unix2texto($texto)

Descrição: Converte padrão Unix para texto. Exemplo: @e@ muda para &.

  • $texto: Texto a ser convertido.

NT2texto($texto)

Descrição: Arruma texto, limpando caracteres NT.

Parâmetros:

  • $texto: Texto a ser arrumado.

grava($grupo, $lingua, $secao, $titulo, $texto)

Descrição: Grava novo registro no NT e retorna ncontrole. A variável $texto deve conter a massa de dados no padrão @/, com os campos especiais da Planilha.

Parâmetros:

  • $grupo: Grupo do registro.
  • $lingua: Língua.
  • $secao: Seção.
  • $titulo: Título.
  • $texto: Texto da massa de dados.

gravaMassa($dados)

Descrição: Grava vários registros de uma vez no PlaceIn. A variável $dados deve ser no padrão: grupo}lingua}secao}titulo}texto}foto|.

Parâmetros:

  • $dados: Dados a serem gravados.

consulta($psql, $parcialsql1, $cache="nao")

Descrição: Pesquisa NTSELECT. Informe a Query no $psql e as variáveis em $parcialsql. O resultado da consulta é gravado em cache no servidor até o PlaceIn ser atualizado.

Parâmetros:

  • $psql: Query SQL.
  • $parcialsql1: Variáveis parciais da SQL.
  • $cache: Se "nao", não utiliza cache.

p_open($arquivo, $conteudo, $modo)

Descrição: Cria ou atualiza um arquivo de texto para armazenar dados. Similar a fopen. O $arquivo deve ser um nome sem a extensão, será incluído .txt automaticamente. $modo deve ser "a" (adiciona o novo $conteudo) ou "w" (apaga e grava o novo conteúdo).

Parâmetros:

  • $arquivo: Nome do arquivo.
  • $conteudo: Conteúdo a ser gravado.
  • $modo: Modo de gravação.

gravaNToc($arquivo, $conteudo, $modo)

Descrição: Similar a p_open(), porém grava um arquivo que não pode ser acessado pelo navegador, nem encontrado pelo Google. Ideal para informações confidenciais.

Parâmetros:

  • $arquivo: Nome do arquivo.
  • $conteudo: Conteúdo do arquivo.
  • $modo: Modo de gravação.

alteranomeNToc($nomeantigo, $nomenovo)

Descrição: Altera o nome de um arquivo NToc.

Parâmetros:

  • $nomeantigo: Nome antigo do arquivo.
  • $nomenovo: Novo nome do arquivo.

leNToc($arquivo)

Descrição: Lê o conteúdo de um arquivo gravado com a função gravaNToc().

Parâmetros:

  • $arquivo: Nome do arquivo a ser lido.

freadNT($filename, $pedaco='', $inicio='', $fim='')

Descrição: Função para ler pedaço de arquivo de texto. Ideal para arquivos grandes. Lê blocos de 512Kb e retorna a parte do texto que inicia com uma string de texto e termina com outra string de texto.

Parâmetros:

  • $filename: Caminho do arquivo.
  • $pedaco: Parte do arquivo a ser lida.
  • $inicio: String de início.
  • $fim: String de fim.

linkbuild($texto)

Descrição: Cria um texto com links para as palavras-chave do cliente. As palavras-chave e URLs devem estar na planilha SEO do domínio K2web.com.br.

Parâmetros:

  • $texto: Texto para gerar links.

alteraNT($ncontrole, $campo, $resultado)

Descrição: Altera valor de planilha especial no PlaceIn. Campos no padrão @/.

Parâmetros:

  • $ncontrole: Controle.
  • $campo: Campos a serem alterados.
  • $resultado: Novos valores.

alteraNTn($ncontrole, $campo, $resultado, $cookieplacein)

Descrição: Modo mais rápido de gravação em massa. Altera valor de planilha especial no PlaceIn.

Parâmetros:

  • $ncontrole: Controle.
  • $campo: Campos a serem alterados.
  • $resultado: Novos valores.
  • $cookieplacein: Cookie do PlaceIn.

alteraNTB($ncontrole, $campo, $resultado)

Descrição: Faz o mesmo que a função alteraNT, porém utiliza um buffer para aumentar a velocidade.

Parâmetros:

  • $ncontrole: Controle.
  • $campo: Campos a serem alterados.
  • $resultado: Novos valores.

alteraNTmassa($ncontrole, $massa)

Descrição: Altera todos os campos @/ presentes em $massa no registro indicado em $ncontrole.

Parâmetros:

  • $ncontrole: Controle.
  • $massa: Massa de dados a serem alterados.

alteraNTmassa3($dados)

Descrição: Alteração de múltiplos registros em alta velocidade. A variável $dados deve ter o padrão: $ncontrole}$nomecampo}$valor|.

Parâmetros:

  • $dados: Dados a serem alterados.

alteraNT2($planilha, $ncontrole, $campo, $resultado)

Descrição: Altera valor de planilha MySQL (mensagens) no PlaceIn.

Parâmetros:

  • $planilha: Planilha (mensagens).
  • $ncontrole: Controle.
  • $campo: Campos a serem alterados.
  • $resultado: Novos valores.

tabdin($resultado, $display, $cabecalho)

Descrição: Imprime resultado na tela, em HTML, de acordo com a opção de visualização.

Parâmetros:

  • $resultado: Resultado da consulta.

desmontalink($massa)

Descrição: Transforma *link( em link HTML.

Parâmetros:

  • $massa: Massa de dados contendo links.

gravamailto($variaveis)

Descrição: Grava e-mail no MailTo. O primeiro parâmetro deve ser o e-mail, o segundo nome, o terceiro empresa e o quarto data de aniversário no formato Unix, o quinto grupo.

Parâmetros:

  • $variaveis: Variáveis de e-mail.

vearquivo($indice, $diretorio, $pesquisaem, $indice2)

Descrição: Verifica quais arquivos têm a palavra indicada na variável índice, no diretório especificado.

  • $indice: Índice da pesquisa.

shortlink($linkmensagem, $pagina, $tipo)

Descrição: Gera um link reduzido. O valor padrão de tipo é "simples" e gera um shortlink inseguro e sequencial. Para um shortlink mais seguro, defina o valor como "seguro".

Parâmetros:

  • $linkmensagem: Link a ser encurtado.
  • $pagina: Página do link.
  • $tipo: Tipo de shortlink ("simples" ou "seguro").

geoip($ips)

Descrição: Verifica a geolocalização de um IP.

Parâmetros:

  • $ips: IP a ser localizado.

NTics($titulo, $DTinicio, $DTfim, $ncontrole, $destinatarios, $mensagem, $assunto)

Descrição: Gera um arquivo padrão ics para importação em agendas do Outlook ou iCal. Será criado um arquivo na pasta biblioteca/fotos/NCONTROLE_agenda.ics e enviado via e-mail.

Parâmetros:

  • $titulo: Título do evento.

  • $ncontrole: Controle da tarefa.
  • $destinatarios: E-mails dos destinatários.
  • $mensagem: Mensagem complementar.
  • $assunto: Assunto do e-mail.

tabdin2($resultado, $display, $cabecalho)

Descrição: Imprime em HTML uma tabela com o resultado fornecido pela pesquisa psql. Igual ao tabdin, porém com recursos JQuery.

Parâmetros:

  • $resultado: Resultado da consulta.
  • $display: Opção de visualização (tabela, tabela2, gráfico, pie, bar, line).
